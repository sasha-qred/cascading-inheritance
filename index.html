CSS Каскад и Наследование

Саша Вербило

The "C" in CSS stands for "Cascade"

Что влияет на каскад

  • Происхождение (и важность)
  • Область видимости
  • Теневое дерево
  • Специфичность
  • Порядок объявления в коде

Приоритетность источника стилей

  • Стили во время CSS-переходов (transition)
  • Браузерные стили с !important
  • Пользовательские стили с !important
  • Авторские (т.е. наши, разработческие) стили с !important
  • Стили во время анимаций
  • Обычные (т.е. без !important) авторские стили
  • Обычные пользовательские стили
  • Обычные браузерные стили

Вычисление значения

  • Объявленные значения (declared values). Все указанные значения.
  • Каскадное значение (cascaded value). Самое приоритетное значение.
  • Указанное значение (specified value). Значение или initial.
  • Вычисленное значение (computed value). Перевод размерных величин в пиксели (чаще всего, но не всегда).
  • Используемое значение (used value). Финальный перевод всего в пиксели.
  • Фактическое значение (actual value). Итог рендеринга.

Свойство all

Формально считается сокращением (shorthand) для всех остальных свойств CSS.

За исключением кастомных свойств, а также (по историческим причинам) свойств, отвечающих за направление текста — direction и unicode-bidi.

Прекрасный CSS reset в одну строчку.

Другие способы влиять на каскад

  • Значение initial.
  • Значение inherit.
  • Значение unset. Наследуемые свойства — inherit, ненаследуемые свойства — initial. Отменяет каскадное значение.
  • Значение revert. Отменяет текущий уровень каскада. Анимации считаются авторскими стилями.
